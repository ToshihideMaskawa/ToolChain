# ToolChain README 

Этот каталог содержит набор инструментов, необходимых для сборки SolidDesigner и генерации кода, упакованных вместе с проектом.
Цель: не зависеть от окружения разработчика и обеспечить воспроизводимую сборку на CI / новых машинах.

## Состав
- `Python310/` : Встроенный Python 3.10 для запуска скриптов генерации
  - Пример: `Python310/python.exe`

## Использование (CMake)
В CMake следует отдавать приоритет встроенному Python.

Пример:
- зафиксировать `Python3_EXECUTABLE` как `.../ToolChain/Python310/python.exe`
- задать это до `find_package(Python3 REQUIRED COMPONENTS Interpreter)`

(Это стабилизирует шаги генерации, например `CommandsConfig.xml` → `SolidDesignerCommands.h`.)

## Примечания
- Рекомендуется выводить сгенерированные файлы в каталог сборки (например, `generated/`) и не хранить их в репозитории.
- Если вы поставляете Windows “embeddable” Python, скрипты могут не работать из-за настроек `python310._pth` и путей к стандартной библиотеке.
  В таком случае проверьте и скорректируйте конфигурацию Python в ToolChain.
